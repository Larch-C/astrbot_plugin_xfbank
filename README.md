# xfbank

xfbank 一个简易的虚拟银行插件


# 支持

[帮助文档](https://github.com/XiaoFu1025/astrbot_plugin_xfbank)
[指令列表]
 "银行操作命令帮助：
           银行操作命令帮助：
            /bank balance - 查询余额
            /bank deposit <金额> - 存款
            /bank withdraw <金额> - 取款
            /bank transfer 本行 <目标卡号> <金额> - 本银行转账
            /bank transfer <目标银行> <目标账户> <金额> - 跨行转账
            /bank record [条数] - 查询交易记录（默认10条，最多20条）
