# xfbank

xfbank 一个简易的虚拟银行插件


# 支持

[帮助文档](https://github.com/XiaoFu1025/astrbot_plugin_xfbank)
[指令列表]
 "银行操作命令帮助：
           银行操作命令帮助：
            "银行操作命令帮助：\n"
            "/bank chaxun - 查询余额\n"
            "/bank qiandao - 每日签到（100~500元，含小数）\n"
            "/bank transfer 本行 <目标卡号> <金额> - 本银行转账\n"
            "/bank transfer <目标银行> <目标账户> <金额> - 跨行转账\n"
            "/bank record [条数] - 查询交易记录（默认10条，最多20条）"
